Pyinit
========

`Pyinit` 用于自动初始化包结构。

```zsh
{{module_name}}
├── LICENSEz
├── README.md
├── conf.py
├── docs
├── {{modue_name}}
├── setup.py
└── test
```

特色
--------

· 基于命令行

· 使用简单

· 使用[Kenneth Reitz](http://kennethreitz.org/)推荐的方案

安装方法
--------

```python
pip install pyinit
```

使用方法
--------

在终端中运行 `pyinit create`, 填写一些信息即可根据信息自动生成。
